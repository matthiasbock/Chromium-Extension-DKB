DKB-Steuermappe
===============

Modifiziert den Briefkasten "Tresor" des Online-Banking der Deutschen Kreditbank zur Verwaltung von steuerlich relevanten Dokumenten.

Dieses Projekt wird unabhängig von und ohne Unterstützung der DKB entwickelt.
Es ist kein Produkt der DKB.
